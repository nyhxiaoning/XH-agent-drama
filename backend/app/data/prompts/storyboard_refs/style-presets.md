用户提供的画风关键词不在预设中时，按以下维度构建完整描述：

1. **视觉基调**：写实 / 风格化 / 动画 / 混合
2. **色彩体系**：主色调 + 辅色 + 对比关系
3. **光线性格**：光源类型 + 方向 + 强度 + 色温
4. **质感/纹理**：皮肤 / 布料 / 背景材质
5. **镜头特征**：焦段感觉 + 景深 + 畸变 + 特殊光学效果
6. **情绪参考词**：最多 **2个** 英文情绪词（cinematic / ethereal / gritty / tender / tense / melancholic 等）

**输出格式要求：**
- 描述控制在 **6组以内**，**最多不超过60个字符**
- 以中文为主，保留专业术语关键词使用英文
