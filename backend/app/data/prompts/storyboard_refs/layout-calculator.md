## 非标准帧数处理规则

帧数不在表中时（如5、7、11、13、14、17–19、21–24帧），取**最近的较大标准帧数**，用空格占位：

```
画布宽 = 单帧宽 × 列数
画布高 = 单帧高 × 行数
最终比例 = 化简后取最接近 GPT Image2 支持列表的比例
```

**示例：**
- 9:16视频，5帧 → 套用6格布局（2行×3列），第6格留空，画布比例 **4:5**
- 9:16视频，11帧 → 套用12格布局（3行×4列），第12格留空，画布比例 **3:4**
- 16:9视频，17帧 → 套用20格布局（4行×5列），后3格留空，画布比例 **5:4**
