## 模块一：情绪基底明细（A–G）

> 情绪基底是每帧的底层情绪定调。先用 index 的快速检索卡 + 整合矩阵取四维起点，再回本模块补 机位/构图/光影/禁忌。

### A 权力与对峙类

#### A1 压迫感（强者施压、被威胁、被控制）
| 维度 | 推荐方案 |
|------|---------|
| 景别 | 受压方：近景～特写；施压方：全景（信息量不对等） |
| 机位 | 施压方仰拍 10–20°；受压方俯拍 10–15° |
| 运镜 | 慢推镜推向受压方（越慢压迫感越强） |
| 焦段 | 施压方用 24–35mm（轻微变形增压迫）；受压方用 85mm |
| 构图 | 施压方居框中上；受压方偏下/边缘；负空间压向受压方 |
| 光影 | 高对比侧光；施压方半脸阴影；受压方顺光或逆光 |
| 节奏 | 长镜不切，让压迫在时间中积累 |
| ⚠️禁忌 | 避免平视平角双人同框；避免对称构图（消解权力落差） |

#### A2 权力展示（霸总登场、身份揭露、震场）
| 维度 | 推荐方案 |
|------|---------|
| 景别 | 远景→全景→近景（层层揭示，仪式感） |
| 机位 | 低机位仰拍 15–30° |
| 运镜 | 慢速环绕 + 慢推；配合人物动作节拍 |
| 焦段 | 85mm（人物美化，背景虚化） |
| 构图 | 人物中心；前景模糊遮挡（人群/道具），从遮挡中浮现 |
| 光影 | 逆光强轮廓光（发际/肩线发光）；低调高对比 |
| 节奏 | 登场前静默 0.5–1s；入场后慢动作 |
| ⚠️禁忌 | 避免俯拍；避免人物置于框架边缘 |

#### A3 针锋相对（正面对峙、撕逼、互不相让）
| 维度 | 推荐方案 |
|------|---------|
| 景别 | 近景正反打为主；穿插大特写眼神 |
| 机位 | 平视；随情绪力量转移动态调整仰/俯角 |
| 运镜 | 固定为主；台词密集时快速切换正反打 |
| 焦段 | 85–100mm（隔离背景，聚焦表情） |
| 构图 | 过肩构图；两人各占三分之一；严格保持轴线一致 |
| 光影 | 双侧光各自相对；整体偏冷 |
| 节奏 | 随台词节奏快切；停顿时定格在弱势方 |
| ⚠️禁忌 | 严禁越轴（180°法则）；避免两人同时运动 |

#### A4 示弱/求饶（跪地、认错、哀求）
| 维度 | 推荐方案 |
|------|---------|
| 景别 | 全景（交代姿态）→ 特写（面部情绪） |
| 机位 | 高角度俯拍示弱方；平视拍强势方反应 |
| 运镜 | 镜头缓慢下移跟随人物下跪 |
| 焦段 | 50mm（客观呈现，不过度美化） |
| 构图 | 示弱方位于画面下三分之一；头顶大量负空间 |
| 光影 | 顶光或逆光（示弱方脸部较暗）；强势方顺光 |
| 节奏 | 慢节奏；给哭泣/颤抖细节留足时间 |

### B 爱情与情感类

#### B1 心动/暧昧（初次心跳、对视、悸动）
| 维度 | 推荐方案 |
|------|---------|
| 景别 | 近景→特写（逐渐拉近）；细节特写（眼睫/嘴唇/手指） |
| 机位 | 平视；轻微仰角拍摄被心动对象 |
| 运镜 | 极缓慢推镜；或固定+浅景深背景虚化 |
| 焦段 | 85–135mm（浅景深奶油虚化） |
| 构图 | 对象居框架中心；周围虚化或暖色光晕包围 |
| 光影 | 暖黄/金色侧光；眼中必须有高光点；柔边逆光 |
| 节奏 | 慢速；主观视角可用镜头轻微抖动表现心跳加速 |
| ⚠️禁忌 | 避免冷色光；避免对称硬光 |

#### B2 温柔爱意（相依、陪伴、深情凝视）
| 维度 | 推荐方案 |
|------|---------|
| 景别 | 中景（两人同框）；近景（单人情绪特写） |
| 机位 | 平视；轻微斜角（视线交汇感） |
| 运镜 | 固定或极轻微漂移（手持温度感） |
| 焦段 | 85mm |
| 构图 | 两人位于画面下三分之二；可用窗框/门框形成框中框 |
| 光影 | 柔和暖光 3000–4000K；无硬阴影；晨光/夕光氛围 |
| 节奏 | 长镜；不急于切换；让停顿成为情绪本身 |
| ⚠️禁忌 | 避免强对比硬光；避免快速剪切 |

#### B3 嫉妒（看见情敌、心理失衡）
| 维度 | 推荐方案 |
|------|---------|
| 景别 | 嫉妒者近景；被嫉妒场景中全景（景别落差制造心理落差） |
| 机位 | 嫉妒者侧后方偷窥角；被嫉妒场景平视 |
| 运镜 | 固定凝视（嫉妒者视角）；手持轻抖 |
| 焦段 | 100–135mm（偷窥感，主体被从远处"拉近"） |
| 构图 | 障碍物/人群做前景遮挡；嫉妒者偏框架边缘 |
| 光影 | 嫉妒者半脸阴影；被嫉妒场景偏暖偏亮（形成对比） |
| 节奏 | 嫉妒者视角长镜；切到主观特写时突然定格 |

#### B4 暗恋/单相思（默默注视、不敢靠近）
| 维度 | 推荐方案 |
|------|---------|
| 景别 | 全景或远景（物理距离=情感距离） |
| 机位 | 平视或轻微高角度（暗恋者视角，略带自卑感） |
| 运镜 | 固定；被暗恋对象轻微虚焦后拉实（主观模糊感） |
| 焦段 | 135mm+（距离即情感） |
| 构图 | 暗恋者在画面角落；被暗恋对象在中心或对侧 |
| 光影 | 被暗恋者偏亮暖；暗恋者偏暗冷 |
| 节奏 | 极缓慢；允许长时间静止镜头 |

### C 冲突与爆发类

#### C1 愤怒爆发（掀桌、摔东西、情绪失控）
| 维度 | 推荐方案 |
|------|---------|
| 景别 | 全景（动作幅度）+ 大特写（青筋/握拳/眼神细节） |
| 机位 | 平视或低机位仰拍（愤怒者更有力量感） |
| 运镜 | 手持轻抖同步情绪；或快速推镜配合爆发瞬间 |
| 焦段 | 35mm（广角给动作更大空间感） |
| 构图 | 不对称；破框感（动作超出画面边界） |
| 光影 | 高对比冷光；大面积阴影；红/橙暖色辅助情绪 |
| 节奏 | 爆发前静止 + 爆发瞬间快切；剪辑节拍感强 |
| ⚠️禁忌 | 避免过于稳定的固定镜头（缺乏情绪张力） |

#### C2 争吵对峙（言语激烈、互相指责）
在 A3 基础上叠加：手持抖动幅度加大；可快速越轴制造混乱感；整体降色温；台词节奏密集时允许越轴

#### C3 崩溃/哭泣（情绪崩塌、痛哭失声）
| 维度 | 推荐方案 |
|------|---------|
| 景别 | 近景→大特写（眼泪/颤抖的手/喉结）；越哭越近 |
| 机位 | 平视或轻微高角度（脆弱感） |
| 运镜 | 固定或极慢推；镜头稳定与内心崩溃形成张力对比 |
| 焦段 | 85mm |
| 构图 | 大量负空间（人物孤立一角）；或极紧密的面部特写 |
| 光影 | 逆光或侧逆光（泪光反射）；整体低亮度；冷蓝调 |
| 节奏 | 长镜；允许 5s+ 沉默；哭泣本身是节奏 |
| ⚠️禁忌 | 避免频繁切换（打断情绪积累）；避免过亮顺光 |

#### C4 震惊/颠覆（得知真相、意外反转）
| 维度 | 推荐方案 |
|------|---------|
| 景别 | 近景（人物反应）→ 快速拉远（孤立感/世界观崩塌） |
| 机位 | 平视→瞬间切低角度（力量对比翻转） |
| 运镜 | 瞬间快速推镜（撞击感）；或快速拉镜 |
| 焦段 | 推进用广角 24mm（夸张变形）；定格用 85mm |
| 构图 | 震惊时人物居中；周围环境对称压迫 |
| 光影 | 冷色调；强光束打亮面部（真相曝光感） |
| 节奏 | 真相揭示前必须有完全静止；揭示瞬间硬切 |
| ⚠️禁忌 | 避免慢推（缺乏冲击力）；避免暖色调 |

### D 悬疑与张力类

#### D1 紧张悬念（等待、未知威胁要来）
| 维度 | 推荐方案 |
|------|---------|
| 景别 | 交替：全景（交代环境威胁）+ 近景（人物反应） |
| 机位 | 轻微偏斜角（不安定感）；偶用主观视角 |
| 运镜 | 极慢推镜（2–4秒推一帧）；或固定静待 |
| 焦段 | 35mm（广角可同时看威胁来源和反应者） |
| 构图 | 大量负空间（未知威胁的空间）；人物偏小 |
| 光影 | 低调高对比；大面积黑暗；仅局部光源 |
| 节奏 | 强调停顿；剪辑点延后（让观众比角色先意识到危险） |
| ⚠️禁忌 | 避免明亮均匀光；避免对称安全构图 |

#### D2 恐惧/惊吓（突然袭击、巨大威胁）
| 维度 | 推荐方案 |
|------|---------|
| 景别 | 特写（受惊者面部）+ 大全景（揭示威胁）；景别极度跳跃 |
| 机位 | 受惊者平视或轻高角度；威胁低角度仰拍 |
| 运镜 | 受惊瞬间 crash zoom 或快速推镜；威胁靠近时跟镜 |
| 焦段 | 24–28mm（广角制造扭曲不安全感） |
| 构图 | 恐惧对象出现在画面边缘/背景角落（先不被注意） |
| 光影 | 极低调；强对比；绿/蓝冷色；阴影面积最大化 |
| 节奏 | 惊吓前长时间静止沉默；惊吓瞬间硬切+音效 |
| ⚠️禁忌 | 避免暖色调；避免均匀曝光 |

#### D3 秘密/监视（偷看、跟踪、被监视而不知）
| 维度 | 推荐方案 |
|------|---------|
| 景别 | 全景或中景（保持距离感）；窥视视角的局部特写 |
| 机位 | 偷窥者主观视角；从障碍物缝隙/边缘拍摄 |
| 运镜 | 手持极轻微抖动（不稳定的偷窥感）；长焦固定 |
| 焦段 | 135–200mm（远距压缩感，像被远处监视） |
| 构图 | 前景有遮挡（门缝/窗框/人群）；主体在中景 |
| 光影 | 偷窥区域暗；被监视者区域亮（对比） |
| 节奏 | 长镜；让观众感受时间漫长 |

#### D4 被揭穿（谎言曝光、身份暴露）
| 维度 | 推荐方案 |
|------|---------|
| 景别 | 揭穿者中景（有力量）；被揭穿者近景→特写（内心崩塌） |
| 机位 | 揭穿者平视或低角度；被揭穿者轻微俯视 |
| 运镜 | 快推至被揭穿者面部；然后固定等待反应 |
| 焦段 | 85mm |
| 构图 | 揭穿者居框架优势位；被揭穿者被"逼"到边缘 |
| 光影 | 被揭穿者被强光照射（无处藏身感）；冷色调 |
| 节奏 | 台词说完立即切到被揭穿者反应；允许长停顿 |

### E 反转与高光类

#### E1 逆袭/得意（反将一军、赢得胜利）
| 维度 | 推荐方案 |
|------|---------|
| 景别 | 全景（反转结果）+ 近景（得意表情） |
| 机位 | 由俯拍→平拍→仰拍（随权力翻转切换机位） |
| 运镜 | 慢速环绕（展示胜利姿态）；配合转身动作的推镜 |
| 焦段 | 85mm |
| 构图 | 逆袭者从画面边缘回归中心；对手退入角落 |
| 光影 | 逆袭后由冷转暖；逆袭者获得顺光/逆光轮廓光 |
| 节奏 | 反转前刻意减慢；反转瞬间快切+音效 |
| ⚠️禁忌 | 避免逆袭后仍用俯拍 |

#### E2 打脸反转（对手被啪啪打脸）
在 E1 基础上叠加：强调对手反应特写（呆滞/错愕）；对比蒙太奇（嘲讽时 vs 狼狈时）；周围人快速切换（群体见证感）

#### E3 高光亮相（王者归来、绝美出场）
| 维度 | 推荐方案 |
|------|---------|
| 景别 | 远景→全景→近景连续推进（层层揭示） |
| 机位 | 低机位仰拍（绝对权威感） |
| 运镜 | 慢速推镜 + 升镜（由下往上扫视人物） |
| 焦段 | 从 35mm（环境）到 85mm（人物特写） |
| 构图 | 人物中心；强对称宏大；前景人群/障碍物待穿越 |
| 光影 | 逆光强轮廓光（发际/肩膀发光）；主体略过曝（神圣感） |
| 节奏 | 音乐卡点；配合 BGM 节拍；慢动作处理 |
| ⚠️禁忌 | 避免平视平角；避免无轮廓光 |

#### E4 决意出发（下定决心、踏上征途）
| 维度 | 推荐方案 |
|------|---------|
| 景别 | 近景（眼神特写）→ 全景（转身出发） |
| 机位 | 平视（决意是自己做出的，平视=平等感） |
| 运镜 | 人物转身后跟镜；或固定+人物走出画框 |
| 焦段 | 50mm（最自然客观，强调真实决意） |
| 构图 | 出发方向留大量空白（前方有路）；背后是已离开的世界 |
| 光影 | 前方光源（出发方向更亮）；逆光走出 |
| 节奏 | 不拖沓；决意后直接行动 |

### F 内心与情绪类

#### F1 孤独/失落（一个人、被抛弃）
| 维度 | 推荐方案 |
|------|---------|
| 景别 | 大远景或远景（人物渺小） |
| 机位 | 高角度俯拍（人物被世界俯视）；或背影平视 |
| 运镜 | 缓慢拉镜（人物越来越小）；或固定静止 |
| 焦段 | 35mm（广角让人物在空间中更孤立） |
| 构图 | 人物偏框架一角；周围大量空白（孤立在空间中） |
| 光影 | 冷蓝调；阴天散射光；无明显光源方向 |
| 节奏 | 极慢；长镜；允许空镜占据大量时间 |
| ⚠️禁忌 | 避免人物居中；避免暖色光；避免快节奏 |

#### F2 悲伤/压抑（内心痛苦、默默承受）
| 维度 | 推荐方案 |
|------|---------|
| 景别 | 近景（面部不哭泣）；细节特写（握紧的手/低垂的眼） |
| 机位 | 平视或轻微高角度 |
| 运镜 | 固定；或极慢推镜 |
| 焦段 | 85–100mm（浅景深，隔绝外部世界） |
| 构图 | 人物偏框架下半部分；头顶空间压低（被压迫感） |
| 光影 | 冷色侧光；大面积阴影；低整体亮度 |
| 节奏 | 慢；重视空气感；可用呼吸声/环境音主导 |

#### F3 回忆/闪回（浮现往事、时光倒流）
| 维度 | 推荐方案 |
|------|---------|
| 景别 | 偏全景（回忆常以旁观者视角呈现） |
| 机位 | 略带飘忽感（记忆的不确定性） |
| 运镜 | 轻微漂移推镜；或手持轻抖 |
| 焦段 | 50–85mm |
| 构图 | 可用边缘渐晕强调回忆感 |
| 光影 | 美好回忆→暖黄金色；痛苦回忆→冷灰；整体降饱和度 |
| 节奏 | 叠化转场进入；不用硬切 |
| ⚠️禁忌 | 避免清晰锐利光感（回忆应有模糊质感） |

#### F4 释然/治愈（放下过去、内心平和）
| 维度 | 推荐方案 |
|------|---------|
| 景别 | 全景（人与环境融合）；或近景微笑 |
| 机位 | 平视（接纳） |
| 运镜 | 极缓慢拉镜（人物融入更大空间） |
| 焦段 | 50mm（最自然，不美化不丑化） |
| 构图 | 黄金分割；人与自然/空间比例均衡 |
| 光影 | 柔和暖光；逆光（新生/转机）；高光晕染 |
| 节奏 | 慢；允许自然景物空镜过渡；呼吸感强 |

### G 日常与过渡类

#### G1 轻松日常
景别中景为主；机位平视随意；运镜手持轻晃（生活感）；焦段 35–50mm；光影均匀自然暖调；节奏正常偏快

#### G2 过渡空镜
孤独/悲伤→雨滴打窗、空旷走廊、路灯下空街；爱情/温柔→两杯并排咖啡、窗帘被风吹动；
紧张/悬疑→走廊尽头阴影、滴落水声；逆袭/高光→朝阳破云、风吹麦田；
运镜固定为主；时长 1–3s

#### G3 时间流逝
延时摄影（云/人流）；或固定镜头内变化（窗外变暗）；叠化/快切；色温从暖到冷或冷到暖

#### G4 环境建立（三镜组合）
1. 大远景/远景（35mm，交代整体环境）
2. 全景/中景（50mm，人物与空间关系）
3. 细节特写（长焦/微距，道具/光影/纹理，定调情绪）

## 类型叠加层明细（H · 动作类型场景）

> **正交叠加层，不是第八个情绪大类。** 情绪基底（A–G）回答"这帧是什么情绪"；类型叠加层回答"这帧是不是动作类型场景，若是该用什么动作镜头骨架"。二者是两条独立的轴：一帧愤怒的打斗 = C1（情绪基底）+ H1/H2（类型叠加）。
>
> **合成规则**：以类型层的 **景别 / 机位 / 运镜 / 构图** 作动作骨架，以情绪基底的 **光影 / 表演 / 节奏** 做情绪着色；二者冲突时——
> **动作可读性 > 情绪渲染**。镜头存在的目的是让观众看清动作的因果（谁打中谁、谁追上谁、子弹从哪来）。碎剪、全程抖动、景别不交代空间——这些会摧毁动作戏，是新手最常犯的错误。

#### H1 功夫片/武打（徒手对打、套招展示、硬桥硬马）
> 港式武打美学：让身体说话。成龙/李小龙原则——用全景长镜完整呈现招式，少切碎剪。

| 维度 | 推荐方案 |
|------|---------|
| 景别 | 全景/远景为主（完整身体动作可读）；中景接受击瞬间；大特写仅用于关键命中（拳/脚/眼神） |
| 机位 | 平视为主（让观众客观看清技术）；起跳/凌空踢用低角度仰拍；扫腿/倒地用轻俯拍 |
| 运镜 | 横移跟镜随对打者移动；固定全景拍完整一组招式；起跳/吊威亚用升降镜；招式间用快速摇/甩镜衔接 |
| 焦段 | 24–35mm（广角容纳全身+空间+动势）；命中定格瞬间可切 85mm 压缩 |
| 构图 | 为出拳/踢腿留足动作空间（破框感）；对决站位左右均衡；双人各占画面一侧形成对抗轴 |
| 光影 | 清晰可见优先（必须看清招式）；硬光勾勒肌肉/动作线条；showcase 段可高调布光 |
| 节奏 | 长镜展示完整套招（一镜到底是武打最高级形态）；快切只用于受击反应；命中瞬间可定格/微慢动作 |
| ⚠️禁忌 | 严禁碎剪抖动遮丑（看不清招式=失败）；避免大特写为主（丢失身体语言）；避免无意义慢动作泛滥 |

#### H2 动作片/打斗（现代格斗、混战、特技、肉搏冲击）
> 好莱坞动作美学：临场感+冲击力。但 John Wick 法则——再凌厉也要让每一刀每一拳有清晰的空间逻辑。

| 维度 | 推荐方案 |
|------|---------|
| 景别 | 中景为主（贴身肉搏的压迫与冲击）；全景交代战场地理；大特写打击瞬间（拳头/被击面部） |
| 机位 | 平视+低角度交替；动态手持；混乱时可轻微倾斜（荷兰角）制造失衡 |
| 运镜 | 手持跟拍（卷入打斗的临场感）；甩镜/快推拉；命中瞬间 crash zoom |
| 焦段 | 28–35mm（空间+动感，广角增强卷入感）；近身缠斗用 50mm |
| 构图 | 不对称/破框；前景遮挡（身处战斗中）；强对角线动势 |
| 光影 | 高对比硬光；现场实用光源（霓虹/火光）；火花/尘土/血雾增加质感颗粒 |
| 节奏 | 快切（卡在命中点上切）；但每个镜头必须落在可读的空间关系上（John Wick 教训）；关键命中可慢动作 |
| ⚠️禁忌 | 避免纯混乱无地理逻辑的抖动碎剪；避免景别全程过远缺乏冲击；避免方位混乱让观众分不清敌我 |

#### H3 枪战（对射、掩体、扫射、狙击）
| 维度 | 推荐方案 |
|------|---------|
| 景别 | 全景（掩体/站位/战场地理）+ 中近景（瞄准/射击动作）+ 大特写（扳机指/弹壳/枪口）；三者交替 |
| 机位 | 平视；英雄用低角度仰拍；越过枪身的 OTS 瞄准视角；掩体后偷拍角 |
| 运镜 | 对峙/瞄准用固定；掩体间转移用快速横移跟拍；甩镜转向新威胁；crash zoom 到枪口/中弹点 |
| 焦段 | 35mm（空间+掩体关系）；85mm 隔离瞄准表情；长焦压缩表现狙击远距 |
| 构图 | 掩体/前景遮挡（cover 作前景）；枪线/视线引导视线流向目标；火力线对角分割；威胁方向留负空间 |
| 光影 | 高对比；枪口闪光作为点缀光（muzzle flash）；逆光烟雾；冷色调金属感 |
| 节奏 | 开火前紧张静默（对峙停顿）→ 爆发快切；英雄高光可慢动作；中弹反应作为节拍标点 |
| ⚠️禁忌 | 严禁越轴混淆敌我方位（180°法则在枪战中尤其致命）；避免持续平稳长镜缺乏紧张；避免不交代掩体关系 |

#### H4 飙车（追车、逃逸、撞击）
> 速度铁律：速度感来自参照物。没有快速掠过的前景（路灯/对向车/护栏），再快也读作静止。

| 维度 | 推荐方案 |
|------|---------|
| 景别 | 全景/远景（车与环境、相对速度）+ 中景（挡风玻璃后驾驶者神情）+ 大特写（踩油门的脚/握方向盘的手/后视镜眼神/时速表） |
| 机位 | 低角度（夸张速度，车显强势）；车载机位（引擎盖/保险杠/侧面吸盘机位）；高空跟拍（无人机/直升机交代路线）；平行追拍 |
| 运镜 | 横移平行跟拍；快速摇/甩镜表现超车掠过；车内手持（驾驶 POV 抖动）；甩镜衔接视角切换 |
| 焦段 | 24–35mm（广角，前景飞速掠过=速度感）；长焦压缩（车距更近更危险、车流堆叠的紧迫） |
| 构图 | 车头朝向引导动势；前景物体快速掠过（电线杆/对向车流）作速度线索；公路对角线 |
| 光影 | 自然光/路灯频闪（掠过的光=运动感）；阳光穿过行道树频闪；夜间霓虹拖影 |
| 节奏 | 快切；车内 ↔ 车外 ↔ 路面 三视角循环；险情瞬间可慢动作；撞击硬切 |
| 速度技巧 | 轻微降帧（undercrank）增强提速感；撞击/翻车用升格（overcrank）慢动作 |
| ⚠️禁忌 | 避免景别全程一致（须内/外/路面交替）；避免无参照物（缺前景掠过物=速度读作静止）；避免方位混乱看不清追逐关系 |

#### H 类过渡空镜（动作戏的"呼吸"与余韵）
> 对照 G2，但动作戏的空镜更短、更"刚"：用于动作爆发之间的喘息、回合切换、或战斗结束后的余味。动态元素（硝烟/蒸汽/火花）配极慢运镜，留余韵更长。

| 类型 | 推荐空镜（任选） |
|------|---------|
| 功夫片/武打（H1）| 晨光中空荡的练功场、墙上静止的兵器架、随风轻晃的沙袋、地面散落的木人桩与脚印、香炉青烟升腾、抱拳行礼的剪影定格 |
| 动作片/打斗（H2）| 满地碎玻璃与翻倒的桌椅、墙面拳印/凹痕、滴血的指节特写、被踢飞悬空的门、雨夜空巷警灯频闪 |
| 枪战（H3）| 缓缓飘散的硝烟、地面滚动的弹壳、墙上密集弹孔、被击碎霓虹招牌迸出的火花、空弹匣落地、冷却中升起白烟的枪管 |
| 飙车（H4）| 空旷公路延伸至地平线、地面焦黑的刹车痕、引擎盖升腾的蒸汽、隧道顶灯飞掠的拖影、后视镜里远去的追兵、加油站霓虹下静止的车 |

| 维度 | 推荐方案 |
|------|---------|
| 运镜 | 固定为主；动态元素（硝烟/蒸汽/火花）配极慢推或慢拉 |
| 时长 | 0.5–2s（动作戏过渡宜短，避免泄掉张力）；战斗结束的余韵空镜可放宽到 2–3s |
| 作用 | ①回合之间的喘息；②战场地理重新交代；③动作结束后的情绪沉淀（常接 F1 孤独 / F4 释然） |

## 模块二：运镜意图库

> 查询方式：由情绪映射表获得初步运镜后，在此确认叙事意图是否匹配

### 完整运镜词典

#### 固定镜头（Static Shot）
| 项目 | 内容 |
|------|------|
| 核心意图 | 客观旁观；让演员表演和场景本身说话 |
| 叙事效果 | 稳定感、冷静感、戏剧留白；观众与角色保持心理距离 |
| 情绪配对 | 悲伤压抑（F2）、孤独（F1）、悬念等待（D1）、对话对峙（A3）、日常（G1） |
| 最佳时机 | 情绪积累期；角色做出重大决定前；对话密集段 |
| ⚠️禁忌 | 动作激烈场景（显呆板）；高光登场（缺乏仪式感） |

#### 慢推镜（Slow Dolly In）
| 项目 | 内容 |
|------|------|
| 核心意图 | 揭示、心理接近、压迫感渐增、情绪聚焦 |
| 叙事效果 | 观众被引导靠近角色内心；距离收缩=情绪升温 |
| 情绪配对 | 压迫（A1）、心动（B1）、悬念（D1）、崩溃前兆（C3）、揭穿（D4） |
| 速度 | 1–4秒移动 10–30cm；越慢压迫感/亲密感越强 |
| 最佳时机 | 情绪转折点前；台词说完后停顿期间；心理变化发生时 |
| ⚠️禁忌 | 欢快日常场景；已完成情绪积累后（改用固定收尾） |

#### 快推镜/撞击推（Crash Zoom / Fast Push）
| 项目 | 内容 |
|------|------|
| 核心意图 | 冲击、惊吓、强调瞬间、视觉暴力 |
| 叙事效果 | 强制将观众注意力锁定到特定细节或表情 |
| 情绪配对 | 震惊（C4）、惊吓（D2）、愤怒爆发瞬间（C1）、被揭穿（D4） |
| 速度 | 0.3–0.8秒完成推进 |
| 最佳时机 | 真相揭示瞬间；惊吓冲击帧；反转高潮点 |
| ⚠️禁忌 | 温柔情感场景；节奏缓慢段 |

#### 拉镜（Dolly Out / Pull Back）
| 项目 | 内容 |
|------|------|
| 核心意图 | 疏远、孤立、揭示更大背景（反转揭示） |
| 叙事效果 | 角色在世界中的渺小；情绪冷却；全貌揭示的惊喜感 |
| 情绪配对 | 孤独（F1）、失落（F1）、震惊后（C4）、释然（F4） |
| 速度 | 缓慢（孤独/释然）；中速（反转揭示） |
| 最佳时机 | 情绪高潮后的冷却；场景结束；揭示角色周围环境 |
| ⚠️禁忌 | 情绪积累期（拉镜会释放张力）；对峙场景 |

#### 水平摇镜（Pan）
| 项目 | 内容 |
|------|------|
| 核心意图 | 跟随视线；展示空间宽度；连接两个物体或人物 |
| 叙事效果 | 建立空间中物体的关系；引导观众视线移动 |
| 情绪配对 | 环境建立（G4）、权力扫视（A2）、视线引导（B1跟随视线） |
| 速度 | 跟随动作=中速；展示环境=慢速；甩镜=极快 |
| 最佳时机 | 人物进入新空间时；视线反应镜头；空间关系建立 |
| ⚠️禁忌 | 情绪激烈对峙（分散聚焦）；避免无目的摇摆 |

#### 垂直摇镜（Tilt Up / Tilt Down）
| 项目 | 内容 |
|------|------|

| 叙事效果 | Tilt Up→揭示宏大/权力；Tilt Down→压迫/发现细节/坠落感 |
| 情绪配对 | A2权力展示（由下到上扫视）；E3高光亮相；场景高大感建立 |
| 速度 | 慢速（仪式感）；中速（跟随动作） |
| 最佳时机 | 高光登场；建筑/场景宏大感建立 |

#### 横移跟镜（Lateral Track / Dolly）
| 项目 | 内容 |
|------|------|
| 核心意图 | 与角色并行；展示平行动作；运动中的观察 |
| 叙事效果 | 代入感强；同步感；让观众"陪伴"角色行进 |
| 情绪配对 | E4决意出发；F4释然行走；G1日常行走 |
| 速度 | 与人物行进速度匹配 |
| 最佳时机 | 人物行走/奔跑场景；强调行动目的感 |
| ⚠️禁忌 | 静止对话场景；情绪停顿场景 |

#### 跟随镜（Follow Shot）
| 项目 | 内容 |
|------|------|
| 核心意图 | 紧密跟随角色；强调目的感和紧迫感 |
| 叙事效果 | 主观代入；追逐感；不停歇的行动感 |
| 情绪配对 | 追逐悬疑（D2）；紧急行动；逃跑场景 |
| 最佳时机 | 追逐；紧急奔跑；主角大步走向目标 |

#### 环绕/弧形镜（Orbit / Arc Shot）
| 项目 | 内容 |
|------|------|
| 核心意图 | 360°展示；强调关系；仪式感；权力展示 |
| 叙事效果 | 全方位审视人物；关系的揭示（两人环绕=关系中心） |
| 情绪配对 | A2权力展示；E3高光亮相；E1逆袭；重要情感时刻（B2） |
| 速度 | 慢速（展示/仪式）；中速（动感/权力） |
| 最佳时机 | 重要人物登场；关键情感转折；胜利时刻 |
| ⚠️禁忌 | 叙事信息密集段（分散注意力）；普通对话 |

| 项目 | 内容 |
|------|------|
| 核心意图 | 解放、希望、脱离、上帝视角过渡 |
| 叙事效果 | 超越当前局面；故事层次提升；结束感/转场 |
| 情绪配对 | F4释然；E4决意出发；场景结束转场；E3高光亮相（上升+仰拍） |
| 速度 | 慢速（情绪）；中速（转场） |

| 叙事效果 | 从宏观进入微观；进入场景/故事 |
| 情绪配对 | D1悬念建立；集数开场；F1孤独（从高处看渺小人物） |
| 速度 | 慢速 |
| 最佳时机 | 集数开场（从环境降入人物）；悬疑氛围建立 |

#### 手持抖动（Handheld）
| 项目 | 内容 |
|------|------|
| 核心意图 | 真实感、不安定、情绪不稳定、主观参与感 |
| 叙事效果 | 纪录片质感；观众"在场"；情绪失控的视觉化 |
| 情绪配对 | C1愤怒爆发；C2争吵；D2恐惧；D3偷窥；B3嫉妒 |
| 幅度参考 | 轻微（±2–5px）=生活真实感；中度（±5–15px）=情绪紧张；剧烈（±15px+）=极度失控 |
| 最佳时机 | 情绪激烈冲突；恐惧追逐；偷窥视角 |
| ⚠️禁忌 | 权力展示（破坏稳定感）；温柔爱意（破坏柔和感） |

#### 甩镜（Whip Pan）
| 项目 | 内容 |
|------|------|
| 核心意图 | 快速转换注意力；时间跳跃；惊喜/惊吓 |
| 叙事效果 | 剧烈视觉冲击；快速时间/空间切换的转场替代 |
| 情绪配对 | C4震惊；快节奏争吵（C2）；时间流逝（G3）；喜剧节拍 |
| 速度 | 极快，0.1–0.3秒完成 |
| 最佳时机 | 反应镜头切换；快节奏剪辑段；惊喜揭示 |
| ⚠️禁忌 | 情绪沉淀段；慢节奏场景 |

#### 变焦推拉（Zoom In / Zoom Out）
| 项目 | 内容 |
|------|------|
| 核心意图 | 区别于推拉镜：变焦改变透视关系，产生焦虑/迷幻/扭曲感 |
| Zoom In 效果 | 聚焦+背景压缩；比推镜更"侵入"；略有不自然感 |
| Zoom Out 效果 | 揭示+轻微变形；反向变焦（人物稳定+背景拉远=眩晕感） |
| 情绪配对 | 焦虑/恐惧（D2）；心理扭曲（惊吓/解离）；复古风格片 |
| 最佳时机 | 刻意制造不适感时；致敬经典镜头时 |
| ⚠️禁忌 | 追求自然感的场景；现代写实风格（变焦感明显过时） |

## 模块三：景别-叙事节奏表

> 查询方式：情绪映射表给出景别后，在此验证与当前叙事节奏位置的匹配度

### 七级景别完整词典

#### 大远景（EWS · Extreme Wide Shot）
| 维度 | 内容 |
|------|------|
| 画面范围 | 人物极小（不足画面高度 1/5）；环境绝对主导 |
| 信息密度 | 极低（人物）；极高（环境/世界） |
| 情绪强度 | 低（直接），但氛围强烈（孤独/宏大/渺小感） |
| 叙事功能 | ①建立地理位置；②表达孤立感和渺小感；③诗意化情绪处理；④场景结束的呼吸 |
| 最佳时长 | 1.5–5s（纯空镜）；0.5–1s（快速建立环境后切进） |
| 节奏特性 | 减速；给观众喘息空间 |
| 短剧注意 | 9:16竖屏时人物更小，谨慎使用；用于情绪过渡空镜，不宜超过3s |
| 情绪配对 | F1孤独、G2过渡空镜、G4环境建立开场 |

#### 远景（WS · Wide Shot）
| 维度 | 内容 |
|------|------|
| 画面范围 | 人物全身可见，占画面高度约 1/3–1/2；环境仍明显 |
| 信息密度 | 中低；人与环境关系清晰 |
| 情绪强度 | 低到中；客观感强 |
| 叙事功能 | ①交代人物在空间中的位置；②展示群体或多人关系；③动作全貌（跌倒/奔跑） |
| 最佳时长 | 1–3s（建立）；2–5s（动作展示） |
| 节奏特性 | 中速；信息建立型 |
| 短剧注意 | 9:16竖屏中WS可用于替代EWS，人物与环境比例更紧凑 |
| 情绪配对 | G4环境建立（第二镜）、A4示弱（交代跪地姿态）、E3高光登场（开场）、B4暗恋 |

#### 全景（FS · Full Shot）
| 维度 | 内容 |
|------|------|
| 画面范围 | 人物全身（头顶留少量空间，脚部刚好入框） |
| 信息密度 | 中；表情+姿态+服装+道具同时可见 |
| 情绪强度 | 中 |
| 叙事功能 | ①展示人物整体姿态（跪/站/转身）；②建立多人物站位关系；③动作完整性 |
| 最佳时长 | 1–3s |
| 节奏特性 | 中速；交代型 |
| 短剧注意 | 竖屏全景时服装/道具细节可见，适合角色登场亮相 |
| 情绪配对 | E3高光亮相（揭示全身）、E4决意出发（转身）、A4示弱（跪地）、A2权力展示 |

#### 中景（MS · Medium Shot）
| 维度 | 内容 |
|------|------|
| 画面范围 | 腰部以上至头顶；手部动作可见 |
| 信息密度 | 高；表情+姿态+手部动作同时可读 |
| 情绪强度 | 中到高 |
| 叙事功能 | ①最通用的对话景别；②展示手部交互；③兼顾情绪与肢体语言 |
| 最佳时长 | 1–5s（对话段） |
| 节奏特性 | 中速；对话驱动 |
| 短剧注意 | 正反打对话约70%使用中景；过多会显单调，需穿插近景/特写 |
| 情绪配对 | A3针锋相对、G1日常对话、B2温柔同框、D4被揭穿（揭穿者） |

#### 近景（MCU · Medium Close-Up）
| 维度 | 内容 |
|------|------|
| 画面范围 | 胸部以上至头顶；表情主导 |
| 信息密度 | 中高；情绪细节清晰 |
| 情绪强度 | 高 |
| 叙事功能 | ①情绪聚焦；②对话中的关键反应镜头；③情绪递进时从中景推入 |
| 最佳时长 | 0.5–3s |
| 节奏特性 | 中快速；情绪递进型 |
| 短剧注意 | 竖屏近景效果极好（占屏比大），是短剧情绪爆发主力景别 |
| 情绪配对 | B1心动、C3崩溃开始段、B3嫉妒（嫉妒者）、D1悬念（反应） |

#### 特写（CU · Close-Up）
| 维度 | 内容 |
|------|------|
| 画面范围 | 肩膀到头顶（或更紧：下巴到额头） |
| 信息密度 | 极高（仅面部情绪） |
| 情绪强度 | 极高 |
| 叙事功能 | ①情绪高峰定格；②揭示微表情；③反应镜头（震惊/哭泣/愤怒） |
| 最佳时长 | 0.5–2s（快节奏）；2–5s（情绪沉淀） |
| 节奏特性 | 快则冲击；慢则沉淀 |
| 短剧注意 | 竖屏特写视觉冲击力极强，是爽感/痛感核心景别；过多会审美疲劳 |
| 情绪配对 | C3哭泣高峰、C4震惊定格、A1压迫（受压方）、E1逆袭（得意表情） |

#### 大特写（ECU · Extreme Close-Up）
| 维度 | 内容 |
|------|------|
| 画面范围 | 局部细节：仅眼睛/仅嘴唇/仅手指/仅某道具 |
| 信息密度 | 极低（整体），极高（该细节） |
| 情绪强度 | 最高（用于高光细节） |
| 叙事功能 | ①道具/细节揭示；②微表情强调；③悬疑线索聚焦；④情绪爆发的极致表达 |
| 最佳时长 | 0.3–1.5s |
| 节奏特性 | 极快冲击或极慢强调 |
| 短剧注意 | 谨慎使用（每集不超过3–5次）；用于绝对高光时刻效果最佳 |
| 情绪配对 | B1心动（眼神光/眼睫）、A1压迫（手握紧/青筋）、D3秘密（关键道具）、C3哭泣（泪珠） |

### 景别叙事节奏速查

| 场景阶段 | 推荐景别序列 | 说明 |
|---------|------------|------|
| 场景开场 | EWS/WS → FS → MS | 由远及近建立空间 |
| 平静对话 | MS ↔ MS | 正反打，均匀节奏 |
| 情绪递进 | MS → MCU → CU | 景别渐近，情绪升温 |
| 情绪高潮 | CU → ECU → CU | 最近处爆发，回落 |
| 对峙升级 | MS → MCU（快切）→ CU（定格） | 切换加快=张力加大 |
| 场景结束 | CU/MCU → WS/EWS | 从情绪回到环境，呼吸 |
| 动作戏 | WS/FS（全貌）→ ECU（细节）→ WS（结果） | 全貌→细节→全貌 |

## 模块四：焦段-心理效果对照

> 查询方式：情绪映射表给出焦段后，在此验证心理效果是否与情绪意图匹配

### 焦段完整词典

#### 超广角（14–20mm）
| 维度 | 内容 |
|------|------|
| 视角 | 107°–94°（视野极宽，透视极度夸张） |
| 空间感 | 空间极度放大；近大远小效果强烈 |
| 人物变形 | 明显（靠近镜头的面部/肢体变形严重） |
| 背景压缩 | 相反，背景被推远、显得更空旷 |
| 景深 | 极深（近景到无限远都清晰） |
| 心理效果 | 扭曲感、迷幻感、空间焦虑、恶意或癫狂感；或极度壮观的空间感 |
| 适用场景 | 恐怖惊吓瞬间；POV主观视角强调；极度癫狂/失控人物；超宽广环境建立 |
| ⚠️慎用 | 近距离拍人物面部（严重变形）；正常对话场景 |

#### 广角（24–28mm）
| 维度 | 内容 |
|------|------|
| 视角 | 84°–75° |
| 空间感 | 强烈空间感；动感；近大远小但不夸张 |
| 人物变形 | 轻微（靠边缘的人物略有变形） |
| 背景压缩 | 较小；背景清晰可读 |
| 景深 | 深（适合展示人物与环境关系） |
| 心理效果 | 动感、临场感、空间感、轻微压迫/疏离感（近距拍摄时）；自由开阔感（远距） |
| 适用场景 | 动作戏；追逐；愤怒爆发（C1）；施压特写（A1施压方）；场景建立 |
| 推荐搭配 | 手持抖动增强临场感；固定镜头强调空间孤立感 |

#### 标准广角（35mm）
| 维度 | 内容 |
|------|------|
| 视角 | 63° |
| 空间感 | 接近人眼视觉，自然真实 |
| 人物变形 | 极轻微（基本忽略不计） |
| 背景压缩 | 少；环境与人物关系自然 |
| 景深 | 中深（人物+部分环境都清晰） |
| 心理效果 | 自然感、真实感、纪实感；旁观者的客观凝视 |
| 适用场景 | 日常生活场景（G1）；悬念构建（D1）；孤独空间感（F1） |
| 推荐搭配 | 自然光；轻手持；街头/室外场景 |

#### 标准镜头（50mm）
| 维度 | 内容 |
|------|------|
| 视角 | 47°（最接近人眼自然视角） |
| 空间感 | 中性；不放大也不压缩 |
| 人物变形 | 无 |
| 背景压缩 | 轻微 |
| 景深 | 中（取决于光圈） |
| 心理效果 | 最客观中立；不美化不丑化；平等感；真相感 |
| 适用场景 | 决意/坚定（E4）；释然（F4）；示弱（A4）；关键真相时刻（D4揭穿者视角） |
| 短剧注意 | 情绪不够强烈时避免单独使用；应搭配运镜和构图补足情绪 |

#### 人像镜头（85mm）
| 维度 | 内容 |
|------|------|
| 视角 | 28.5° |
| 空间感 | 背景轻微分离；人物突出 |
| 人物变形 | 无（最佳人像焦段，面部最自然） |
| 背景压缩 | 中等（背景虚化，人物与背景分离） |
| 景深 | 浅（大光圈时背景奶油虚化） |
| 心理效果 | 美化感、亲密感、电影感；观众与人物心理距离适中；既近又有尊重感 |
| 适用场景 | 几乎所有情绪场景通用；心动（B1）、崩溃（C3）、对峙（A3）、高光（E3）等 |
| 推荐程度 | ⭐⭐⭐⭐⭐ 短剧最推荐焦段，约70%情绪场景首选 |

#### 中长焦（100–135mm）
| 维度 | 内容 |
|------|------|
| 视角 | 24°–18° |
| 空间感 | 压缩明显；前后层次被拉近 |
| 人物变形 | 轻微压平（有时可使五官更立体） |
| 背景压缩 | 强；背景与人物拉近，背景清晰度下降 |
| 景深 | 很浅（大光圈时背景大幅虚化） |
| 心理效果 | 情感距离感（观察而非参与）；偷窥感；心理隔离；背景压迫感 |
| 适用场景 | 偷窥/监视（D3）；嫉妒（B3）；暗恋（B4）；人物与环境疏离感（F2） |
| 短剧推荐 | 9:16竖屏使用135mm时浅景深效果极好；适合高质感情绪镜头 |

#### 超长焦（200mm+）
| 维度 | 内容 |
|------|------|
| 视角 | 12°以下 |
| 空间感 | 极度压缩；远近景叠加在一起 |

| 背景压缩 | 极强；背景与人物几乎在同一平面 |
