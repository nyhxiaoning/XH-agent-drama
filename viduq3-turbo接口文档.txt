Modelink API · 视频

Copy page

Modelink 视频生成 API，涵盖 OpenAI 风格、Fal 队列、可灵、豆包 Seedance、Vidu、Veo 等多厂商的文生视频、图生视频与任务查询能力。所有请求均需通过 Bearer Token 进行身份认证。

Version 1.0.0

创建参考生视频任务
POST

https://api.qnaigc.com
/queue/fal-ai/vidu/q3/reference-to-video/turbo


使用 Vidu Q3 Turbo 模型创建参考生视频任务。

API 路径带 /turbo 后缀，服务端内部模型 ID 为 viduq3-turbo。 支持 1 到 7 张参考图片，不支持 reference_video_urls。

Query parameters
Name
fal_webhook
Type
string<uri>
Description
可选。HTTPS 回调地址，优先级高于请求体中的 callback_url，需通过 SSRF 校验。

Request body
Body properties
prompt
string, required
视频生成提示词，Vidu Q3 系列最长 2000 字符。

reference_image_urls
string[], required
参考图片 URL 列表，支持 HTTP(S) 图片 URL 或 data:image/* Base64 图片。 Vidu Q3 / Q3 Turbo 参考生视频支持 1 到 7 张参考图片。

duration
integer
视频时长，单位秒；未传时默认 5 秒，取值范围为 3 到 16。

default: 5; minimum: 3; maximum: 16


resolution
enum
输出分辨率，Vidu Q3 系列默认 720p。

default: 720p


aspect_ratio
enum
视频宽高比；Vidu Q3 系列支持 16:9、9:16、1:1、4:3、3:4。

default: 16:9

seed
integer
随机种子；不传或传 0 时由系统自动生成。


movement_amplitude
enum
运动幅度。

default: auto

bgm
boolean | null
是否添加背景音乐；不传表示未指定。

audio
boolean | null
是否音视频直出；不传表示未指定。

Request
POST
/queue/fal-ai/vidu/q3/reference-to-video/turbo

Shell

cURL

Copy
curl 'https://api.qnaigc.com/queue/fal-ai/vidu/q3/reference-to-video/turbo?fal_webhook={fal_webhook}' \
  --request POST \
  --header 'Authorization: Bearer {token}' \
  --header 'Accept: application/json' \
  --header 'Content-Type: application/json' \
  --data '{
  "prompt": "A character walking through a beach catching an apple.",
  "reference_image_urls": [
    "https://storage.googleapis.com/falserverless/web-examples/vidu/new-examples/reference1.png",
    "https://storage.googleapis.com/falserverless/web-examples/vidu/new-examples/reference2.png",
    "https://storage.googleapis.com/falserverless/web-examples/vidu/new-examples/reference3.png"
  ]
}'

创建图生视频任务
POST

https://api.qnaigc.com
/queue/fal-ai/vidu/q3/image-to-video/turbo


创建图生视频任务，返回任务ID和查询任务状态及结果的URL

Query parameters
Name
fal_webhook
Type
string<uri>
Description
可选。HTTPS 回调地址，优先级高于请求体中的 callback_url，需通过 SSRF 校验。

Headers
Name
Content-Type
Type
string
Description
Operation parameter.
Request body
Body properties
prompt
string
文本提示词 生成视频的文本描述。 注：字符长度不能超过 2000 个字符

image_url
string, required
首帧图像 模型将以此参数中传入的图片为首帧画面来生成视频。 注1：支持传入图片 Base64 编码或图片URL（确保可访问）； 注2：只支持输入 1 张图； 注3：图片支持 png、jpeg、jpg、webp格式； 注4：图片比例需要小于 1:4 或者 4:1 ； 注5：图片大小不超过 50 MB； 注6：请注意，http请求的post body不超过20MB，且编码必须包含适当的内容类型字符串，例如： data:image/png;base64,{base64_encode}

audio
boolean
是否使用音视频直出能力，默认为false，枚举值为：

false：不需要音视频直出，输出静音视频
true：需要音视频直出，输出带台词以及背景音的视频 注1：该参数为true时，voice_id参数才生效 注2：该参数为true时，仅q3模型支持错峰 注3：当model 为q3 时，该参数默认值为true
is_rec
boolean
是否使用推荐提示词

true：是，由系统自动推荐提示词，并使用提示词内容生成视频，推荐提示词数量=1
false：否，根据输入的prompt生成视频
seed
integer
随机种子 当默认不传或者传0时，会使用随机数替代 手动设置则使用设置的种子

duration
integer
视频时长参数 viduq3-turbo 默认为 5，可选：1 - 16

resolution
string
分辨率参数 viduq3-turbo 1-16秒：默认 720p，可选：540p、720p、1080p

movement_amplitude
string
运动幅度 默认 auto，可选值：auto、small、medium、large

Request
POST
/queue/fal-ai/vidu/q3/image-to-video/turbo

Shell

cURL

Copy

  --request POST \
  --header 'Authorization: Bearer {token}' \
  --header 'Accept: application/json' \
  --header 'Content-Type: application/json' \
  --data '{
  "prompt": "这个人在跑马拉松",
  "image_url": "https://aitoken-public.qnaigc.com/example/generate-video/running-man.jpg",
  "seed": 2,
  "duration": 4,
  "resolution": "720p",
  "movement_amplitude": "auto"
}'

movement_amplitude
